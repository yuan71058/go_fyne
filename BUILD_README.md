# Fyne应用程序构建指南

本项目提供了多种方式来构建和运行Fyne GUI应用程序。

## 批处理文件说明

### 1. build-gui.bat
快速构建GUI版本（无控制台窗口）的批处理文件。

**使用方法：**
```bash
.\build-gui.bat
```

**功能：**
- 编译生成无控制台窗口的GUI程序
- 输出文件：`fyne-app-gui.exe`
- 适合作为最终发布版本

### 2. build.bat
功能完整的构建工具，提供菜单选择。

**使用方法：**
```bash
.\build.bat
```

**功能选项：**
1. 构建GUI版本（无控制台窗口）
2. 构建控制台版本
3. 运行GUI版本
4. 运行控制台版本
5. 清理构建文件
6. 退出

## 技术说明

### GUI版本与控制台版本的区别

**GUI版本（fyne-app-gui.exe）：**
- 使用 `-ldflags="-H=windowsgui"` 编译标志
- 运行时不显示控制台窗口
- 适合作为最终用户使用的发布版本
- 双击即可运行，用户体验更好

**控制台版本（fyne-app.exe）：**
- 标准编译，无特殊标志
- 运行时会显示控制台窗口
- 适合开发和调试阶段
- 可以查看控制台输出的调试信息

### 环境变量设置

两个版本都使用以下环境变量：
- `CGO_ENABLED=1`：启用CGO（Fyne需要）
- `GOOS=windows`：目标操作系统为Windows
- `GOARCH=amd64`：目标架构为64位

## 使用建议

1. **开发阶段**：使用控制台版本，方便查看调试信息
2. **测试阶段**：使用GUI版本，模拟最终用户体验
3. **发布阶段**：使用GUI版本，提供专业的用户体验

## 注意事项

1. 确保已安装Go开发环境
2. 确保已安装CGO所需的C编译器（如TDM-GCC或MinGW-w64）
3. 如果编译失败，请检查Go环境配置和依赖项
